# `read.data.files`

Read all standard data SAM files and return a list as created by 'setup.sam.data'


## Description

Read all standard data SAM files and return a list as created by 'setup.sam.data'


## Usage

```r
read.data.files(dir = ".")
```


## Arguments

Argument      |Description
------------- |----------------
`dir`     |     Directory to read from


## Details

Read all standard SAM data files


## Value

list (as created by 'setup.sam.data')
